# AI-Based Internship Recommendation Engine  
### PM Internship Scheme

An AI-powered internship recommendation system designed to match students with the most relevant internships under the **PM Internship Scheme**.

The system analyzes student profiles, academic performance, skills, and career interests to generate **personalized internship recommendations**, helping improve employability and learning outcomes.
